# ERC
El presente repositorio contiene el software que es desarrollado para el estudio de la identificación de biomarcadores alternativos para la detección de la enfermedad renal crónica (ERC), mediante el procesamiento de un conjunto de datos depurados y la aplicación de los algoritmos K-Nearest Neighbors (KNN), Random Forest y regresión logística.

This repository contains the software that is developed for the study of the identification of alternative biomarkers for the detection of chronic kidney disease (CKD), through the processing of a cleansed data set and the application of the K-Nearest Neighbors (KNN), Random Forest and logistic regression algorithms.
